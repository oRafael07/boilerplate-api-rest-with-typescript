# Installation

```bash
// Clone Project
$ git clone <https://github.com/oRafael07/boilerplate-api-rest-with-typescript>
```

```bash
// Install Packages
$ yarn install
```
```bash
// Run migrations
$ yarn prisma migration dev
```
```bash
// Scripts available
$ yarn dev
$ yarn build
$ yarn start (use build)
```
